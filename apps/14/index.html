    ∞ ARR-14: Web

    General

    Web is the API that supports the project website. Current (3rd) version is using FastAPI framework.

    Flow Diagrams

    People

    flowchart TD
    classDef success fill:#376622

    subgraph init_service [Init People service]
    direction LR
        init_db_session(Database session) -.- init_storage_client(Storage client)
    end

    subgraph apply_filters [Apply filters to dataset]
    direction LR
        apply_age(Age)
        apply_alive(Alive or not)
        apply_sex(Gender)
        apply_flags(Nationality)
        apply_expertise(Expertise)
        apply_groups(Custom groups)

        apply_age --> apply_alive --> apply_sex --> apply_flags --> apply_expertise --> apply_groups
    end

    endpoint(Trigger people endpoint)
    get_ppl(Fetch all relevant patients)
    apply_sorting(Apply sorting to dataset)
    paginate(Paginate dataset)
    fetch_pics(Fetch thumb URLs for requested page)
    combine(Merge thumbs URLs into response)
    return((Return<br>requested<br>page)):::success

    endpoint --> init_service --> get_ppl --> apply_filters --> apply_sorting --> paginate --> fetch_pics --> combine --> return

    Theory

    flowchart TD
    classDef success fill:#376622

    subgraph init_service [Init Theory service]
    direction LR
        init_db_session(Database session)
    end


    subgraph sequence [Sequence]
        get_sequences(Fetch all sequences)
        get_sequence_for_display(Get sequence for display)
        get_sequence_data(Fetch relevant notes)

        get_sequence_for_display --> get_sequence_data
    end

    subgraph apply_filters [Apply filters to notes dataset]
        filter_date(Date)
    end

    subgraph identify_secondary [Collect adjacent & sibling tags]
    direction LR
        extract_adjacent(Extract other tags from<br>notes dataset ~ adjacent) -.- fetch_sibling(Fetch other tags of<br>the same parent ~ siblings)
    end

    subgraph tag [Tag]
        get_tag_for_display(Get tag for display)
        get_tag_data(Fetch relevant notes)
        get_tags(Fetch all tags)

        get_tag_for_display --> get_tag_data --> identify_secondary
        get_tags --> identify_secondary
    end

    endpoint(Trigger Theory endpoint)
    apply_sorting(Apply sorting to notes dataset)
    paginate(Paginate notes dataset)
    shape_additional(Shape object with additional data)
    return((Return<br>requested<br>page)):::success

    endpoint --> init_service -- if sequence_id --> sequence
    init_service -- if no sequence_id --> tag
    sequence & tag --> apply_filters --> apply_sorting --> paginate --> shape_additional --> return

    Search

    flowchart TD
    classDef success fill:#376622

    subgraph init_service [Init Search service]
    direction LR
        init_db_session(Database session) -.- init_static(Set list of allowed entities)
    end

    subgraph lookup [Perform lookup]
        set_entities(Decide which entities to search in)
        lookup_basic(Do basic lookup)
        lookup_expand(Expand initial lookup<br>with additional data)

        set_entities -- for each entity --> lookup_basic -- for each result --> lookup_expand
    end

    endpoint(Trigger Search endpoint)
    return((Return<br>combined<br>results)):::success

    endpoint --> init_service --> lookup --> return
