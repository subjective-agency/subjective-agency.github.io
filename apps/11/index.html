    ∞ ARR-11: Transsuply & Transsuply2

    General

    Transsuply is application which purpose is to supply the Transfactory with audio files to be transcribed. The first generation (Transsuply) is supposed to be started periodically, and works over a local directory with media already downloaded and processed. The second (Transsuply2), is meant to run constantly; it picks up downloading jobs from the queue table, downloads the media, extracts audio if required, and then proceeds to upload it to the storage same as in the original flow.
